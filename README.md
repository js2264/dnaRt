## Installation

You can install the released version of dnaRt from [Github](https://github.com/js2264/dnaRt) with:

``` r
remote::install_github("js2264/dnaRt")
```

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(dnaRt)
```
